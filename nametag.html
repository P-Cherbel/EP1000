Create a nametag

This nametag was created using fusion 360.

Steps:

Create a sketch.

On the origin, click one of the planes and then select “create a sketch” on the tab. Then, I set up the name tag sketch using the dimensions in the assignment. Once the sketch is drawn, click finish sketch in the bar.

Extrude.

To make the 3D shape, I selected the plane of the entire drawing except the hole. Click extrude from the bar and then set the start as the profile plane and the set the direction as symmetric. The distance was set to 10mm.

Cut.

To make the well where the name will sit, select the box from the drawing in step 1 and then click extrude. The operation should be set as cut and the direction be from profile plane.

Draw your name

Select the plane of the front of the name tag and then create a sketch from plane. Insert text and then write your name. Click finish sketch.

Extrude name.

Select the plane of the name and then extrude. This time make sure the operation is “join”.

Tip

Write down the dimensions required so that you can plan out the sketches.
