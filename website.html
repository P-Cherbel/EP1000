Jake Wright Tutorial

For this, I followed a video made by Jake Wright on learning CSS.

Steps:

Make a HTML website

I used the HTML website I created from an earlier class introducing myself.

It has 4 parts:

  • Header
  • Navigation
  • About me
  • Contact

Sketch the Layout

Following the video, I drew out on a piece of paper how the site layout should be like.

Label the HTML

I named the different parts of the website using div id=”name of part”

Write out CSS

For every div I made, I wrote it out in CSS using #[div id] and then within the brackets, put in information like the size (especially for the container div which contains everything).

For example, I indicated the width of the container to be 1024

To make it easier, I followed the advice from the video to draw out and plant the layout of the website

Tip

Make sure all of the divisions are set up correctly. I had the problem of having the div not encompassing the entire section so always check.
